About Football Hub

A lightweight, static football site you can host on GitHub Pages. No frameworks, no build step.

What’s inside

  • Responsive multi-page HTML.
  • Clean CSS with dark theme and accessibility basics.
  • Vanilla JS that renders fixtures, table, and scorers from JSON.
  • Ready for GitHub Pages: just push to a repo and enable Pages.

How to customize

  1. Edit assets/data.json with your real teams and fixtures.
  2. Tweak colors in assets/styles.css under :root.
  3. Add pages by copying an existing HTML and updating the nav.
